# Countries Analysis with Pandas
I am doing a project that demonstrates the countries data. It is about exploring data with Python, Pandas and Jupyter Notebook. It includes population, regions, continents, capitals, democracy scores, political leaders and official country names.

## Project Overview
The goal of this project is to practice working with data using Pandas. I am using a dataset about countries. Answering questions from the data. The notebook has 194 rows and 64 columns. I am looking at the data filtering it sorting it and doing text analysis. I am also checking for missing values.
